你是一個專業的全端架構師，任務是設計並建立一個名為 Fit 的運動與生活紀錄平台 DEMO。

需求如下：
1. 平台定位：
   - 提供個人、健身房、企業健康管理的運動紀錄與分析。
   - 整合穿戴裝置、手機應用、IoT健身器材。

2. 功能模組：
   - 運動資料收集：跑步配速、騎乘功率、重訓肌力、游泳泳姿、有氧卡路里、心率、睡眠、步數。
   - 雲端同步模組：保存紀錄，支援長期追蹤與跨裝置存取。
   - 數據分析與視覺化：心率區間、速度、瓦特、步頻、象限分析、長期報告。
   - 團體訓練系統：多人即時數據收集，教練端分析。
   - 健康管理：體脂肪、肌肉率、水分率、睡眠品質。
   - 群組系統：成員運動數據分析、球隊/企業健康推動。

3. 技術架構：
   - 前端：Vue 3 + Vite + Pinia (狀態管理) + ECharts/D3.js (數據視覺化)
   - 後端：Node.js + Express (REST API + 數據分析邏輯)
   - 資料庫：Supabase (PostgreSQL + Realtime + Auth + Storage)
   - IoT 整合：藍牙 API、感應器資料收集
   - 雲端：抽象化雲端服務 (可替換 AWS/Azure/GCP/Supabase)，支援同步與報告生成

4. 輸出：
   - 系統架構圖
   - API 設計 (運動紀錄、群組管理、數據分析)
   - 前端 UI 規劃 (運動報告、群組分析、健康追蹤)

請逐步規劃並生成建站方案，包含技術選型、模組設計、數據流程、前後端架構。
